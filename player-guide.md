> 欢迎来到 LegacyLand！这里是一个以国家为核心的文明模拟服务器。

## 1. 刚进服务器：了解自己的状态

进入服务器后，你有几个需要关注的生存属性，它们比原版更复杂一点。

### 查看个人状态

```
/player info
/status info
```

你会看到：

| 属性 | 说明 |
|------|------|
| 生命值 | 普通血量，被打掉了要喝药 |
| 食物 | 饿了会掉血，要吃东西 |
| 水分 | 干旱地区会脱水，要喝水 |
| 体温 | 沙漠太热、雪地太冷，都会影响你 |
| 职业 | 你选择的主职业和副职业 |

> **贴士**：体温系统是动态的。当前是什么季节、你在什么地形，都会影响体温。
> 用 `/season info` 查看当前季节和气候。

## 2. 加入一个国家

LegacyLand 以 **Towny 国家体系** 为基础。加入一个国家非常重要，因为大多数经济和政治功能都需要你是某个国家的成员。

### 怎么加入？

加入国家需要找到该国的城镇长（Mayor）邀请你，或者通过 Towny 的 `/nation join <国家名>` 等方式加入。一旦加入，你就有了：

- 国家货币账户（可以存钱、取钱）
- 参与投票和法令的资格
- 参与外交和战争的资格

### 查看自己在国家里的身份

```
/legacy info
```

这里会显示你的国家、当前政体、以及你被分配的角色（例如：将军、外交官、商人等）。

## 3. 选择职业

职业分为**主职业**和**副职业**，会影响你的技能成长和某些系统的加成。

```
/player profession main <职业名>
/player profession sub <职业名>

```

## 4. 银行与货币：如何赚钱和花钱

LegacyLand 有独立的货币体系。每个国家都有自己的货币，货币以**实体物品**的形式存在，可以存入银行变成电子余额。

### 货币基本操作

**存款**：手持货币物品，输入：
```
/bank deposit
```
货币会消失，换成你银行账户里的数字余额。

**取款**：从账户提取货币物品：
```
/bank withdraw <金额> [国家名]
```
如果你只在一个国家有余额，国家名可以省略。

**查余额**：
```
/bank balance
/bank balance <国家名>
```

**转账给别人**：
```
/bank transfer <玩家名> <金额> [国家名]
```

**兑换不同国家的货币**（收 15% 手续费）：
```
/bank exchange <金额> <源国家> <目标国家>
```

> **贴士**：货币汇率由各国的国库储备量决定，不是固定的。强大的国家货币更值钱。

---

### 贷款

手头紧？可以向国家申请贷款：
```
/finance loan <金额> [国家名]
```

查看你的贷款：
```
/finance loans
```

还款：
```
/finance repay <贷款ID> <金额>
```

> **注意**：贷款有利息，逾期可能影响你的信用评分。

## 5. 市场与贸易：摆摊和买东西

### 玩家市场

每个城镇可以设立市场。在市场里，玩家用**售货箱**出售物品。

**找到市场**：在市场区域走一走，找有售货箱的地方。

**给售货箱定价**（箱主操作）：
```
/price set <金价>
```
输入命令后，**点击**你想设置价格的箱子即可。

**查看市场信息**（站在市场区域）：
```
/market info
```

---

### 资源与运输

服务器有资源区域系统，不同地区产出不同资源。

查看你身上的资源重量和当前区域：
```
/resource info
```

计算运输到某个坐标的费用：
```
/resource transport <x> <y> <z>
```

## 6. 期货合约：提前预约物资

期货合约是一种「提前约定」的交易方式。卖方现在挂出合约，买方购买后，卖方在约定时间内交货。

### 作为卖方：挂出期货

手持你想出售的物品，然后：
```
/finance future <物品> <数量> <价格> <天数> [国家]
```
例如：`/finance future diamond 64 500 7` 表示 7 天内交付 64 颗钻石，总价 500。

### 作为买方：购买期货

查看市场上的期货合约：
```
/finance futures open
```

购买某个合约：
```
/finance buy <合约ID>
```

到期后领取货物：
```
/finance claim <合约ID>
```

### 作为卖方：履约交货

准备好物品后，手持物品：
```
/finance deliver <合约ID>
```

## 7. 股票市场：投资国家

每个国家可以上市一支股票。股价与该国的**实力**（国库储备 + 领土 + 人口）动态挂钩——国家越强大，股价越高。

### 如何买股票

查看当前所有上市股票：
```
/stock list
```

查看某支股票详情：
```
/stock info <股票ID>
```

买入股票（从国家直售或者从卖单撮合）：
```
/stock buy <股票ID> <数量>
```

### 如何卖股票

挂出卖单：
```
/stock sell <股票ID> <数量> <每股价格>
```

撤回卖单：
```
/stock cancel <卖单ID>
```

### 分红

如果你持有某支股票，当该国领导人发放分红时，你会自动收到打入银行账户的分红款。

### 股价是怎么变的？

```
股价 = 发行价 × (当前实力评分 / 发行时实力评分)

实力评分 = 国库储备 × 0.5 + 领土区块数 × 200 + 人口数 × 500
```

国家打胜仗、扩张领土、吸引更多成员 → 股价上涨。
国家萎缩、战败、人口流失 → 股价下跌。

## 8. 外交：国家之间的关系

### 查看外交关系

```
/diplomacy info
/diplomacy info <国家名>
```

这会打开一个界面，显示你的国家与其他国家的关系状态（盟友、贸易协定、中立、敌对等）。

### 外交关系类型

| 关系 | 效果 |
|------|------|
| 盟友 | 可以互相支援战争 |
| 贸易协定 | 降低货币兑换手续费 |
| 科技协定 | 可以共享科技研究 |
| 中立 | 默认状态，互不干涉 |
| 敌对 | 可以宣战 |

### 发起外交行动（需要领导人权限）

```
/diplomacy ally <国家名>      —— 申请结盟
/diplomacy trade <国家名>     —— 签订贸易协定
/diplomacy tech <国家名>      —— 签订科技协定
/diplomacy neutral <国家名>   —— 改为中立
```

### 保护担保

如果你的国家想保护一个弱小国家，可以建立保护担保：
```
/guarantee add <国家名>
```

查看谁在保护你、你在保护谁：
```
/guarantee list
```

## 9. 战争：打仗怎么打

LegacyLand 有多种战争模式，从外交宣战到实地攻城，层次丰富。

### 经济制裁（打仗前的手段）

不想直接开战？可以先用经济手段施压：

```
/ecowar embargo <国家名> <天数>     —— 资源禁运
/ecowar sanction <国家名> <关税率>  —— 经济制裁（提高对方货币的关税）
```

查看经济数据排行：
```
/ecowar gdp          —— 各国 GDP 排名
/ecowar stats [国家] —— 查看经济数据
```

### 宣战与普通战争

```
/diplomacy war <国家名>    —— 宣战
/war start <城镇名> <战役名>  —— 发起对某城镇的战争（需要双方相邻）
/war list                 —— 查看当前所有战争
/war info                 —— 查看自己参与的战争详情
```

加入战争：
```
/war join <战争ID> <角色>   —— 角色：soldier（士兵）/ logistics（后勤）/ scout（侦察）
```

投降 / 停战：
```
/war surrender   —— 投降（需要领导人权限）
/war peace       —— 求和（需要领导人权限）
```

### 攻城战（Siege War）

攻城战是占领敌方城镇的方式，需要在实地进行。

**建立前哨站**（站在信标上）：
```
/siege outpost
```

**建立补给站**（站在信标上）：
```
/siege station create
```

**添加城市核心目标**：
```
/siege core add <核心名>
```

查看攻城战进度：
```
/siege info
```

### 旗帜战（Flag War）

旗帜战是一种轻量级的领土争夺方式。

```
/flagwar info    —— 查看自己的旗帜战信息
/flagwar list    —— 查看所有旗帜战
/flagwar cancel  —— 取消自己发起的旗帜战
```

## 10. 国家领导人专区

> 以下内容需要国家领导人（国王/总统）权限。

### 建立国家经济

**第一步：创建国库**
```
/economy create
```

**第二步：铸造货币**（消耗金锭，产出货币物品）
```
/economy mint <面值>
```

**查看国库状态**：
```
/economy info
```

---

### 政体与政治系统

选择你的国家政府形式：
```
/legacy government FEUDAL     —— 封建制
/legacy government REPUBLIC   —— 共和制
```

切换政治系统（有冷却时间）：
```
/legacy politics set <政体ID>
/legacy politics list           —— 查看所有可用政体
```

给成员分配角色：
```
/legacy role <玩家名> <角色>
```

---

### 法令系统

法令可以修改国家内的规则（如调整税率、特殊政策等）。

**直接颁布**（无需投票）：
```
/law enact <类型> [小时] [参数]
```

**提案投票**：
```
/law propose <类型> [小时] [参数]   —— 发起提案
/law vote <提案ID> yes/no           —— 成员投票
```

查看当前法令和提案：
```
/law list [国家]
/law proposals [国家]
```

废除法令：
```
/law repeal <法令ID>
```

---

### 科技研究

科技树会解锁国家的额外能力。查看可研究的内容：
```
/tech list
/tech info <科技ID>
/tech status
```

研究科技（消耗研究点数）：
```
/tech research <科技ID>
```

---

### 发行股票

你的国家可以上市一支股票（每国只能有一支）：
```
/stock issue <ID> <显示名称> <总股本> <发行价>
```

例：`/stock issue china 大汉股份 10000 100` 发行 1 万股，每股 100。

**发放分红**（从国库扣款，自动打给所有持股玩家）：
```
/stock dividend <每股金额>
```

**股票下市**：
```
/stock delist <股票ID>
```

---

### 经济制裁与禁运

```
/ecowar embargo <目标国家> <天数>      —— 资源禁运
/ecowar lift <目标国家>                —— 解除禁运
/ecowar sanction <目标国家> <关税率>   —— 经济制裁（0.1 = 10%关税）
```

### 加入战争步骤

1. `/war list` — 找到战争 ID
2. `/war join <ID> soldier` — 以士兵身份加入
3. 实地作战或参与攻城

### 新玩家入门顺序

```
1. /player info       → 看看自己的属性
2. 加入一个国家
3. /player profession → 选择职业
4. /bank balance      → 看看有没有初始余额
5. /stock list        → 研究一下各国股票
6. /diplomacy info    → 了解当前外交格局
7. /war list          → 看看有没有在进行的战争
```
